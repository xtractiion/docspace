## ğŸš€ Getting Started

### Prerequisites

- Flutter SDK 3.10+
- Android Studio 2023.1+
- Java 17

### Installation

1. **Clone the repository**
   ```sh
   git clone https://github.com/xtractiion/docspace.git
   ```

2. **Open in Terminal/VS Code**
   - Navigate to the directory
   ```sh
   cd docspace
   ```

3. **Get Dependencies**
   ```sh
   flutter pub get
   ```

4. **Run**
   - Connect a device
   ```sh
   flutter run --release
   ```
